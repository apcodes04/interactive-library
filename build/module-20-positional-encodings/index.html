Positional Encodings in Transformers

  1. Step 1: Drag the "Token position" slider (pos 0 to 50).
  2. Step 2: Watch the vertical marker line intersect all 4 sinusoidal curves.
  3. Step 3: Read the 4-element positional vector output PE(pos).
PE(pos, 0) = sin(pos/1)
PE(pos, 1) = cos(pos/1)
PE(pos, 2) = sin(pos/10)
PE(pos, 3) = cos(pos/10)
Positional Vector PE(pos) [d=8, i=0..1]:
dim 0 (sin0): 0.000
dim 1 (cos0): 1.000
dim 2 (sin1): 0.000
dim 3 (cos1): 1.000

Drag slider to calculate unique multi-frequency sinusoidal positional encodings.

Every position gets its own unique combination of these wave values — that combination is the position, encoded as numbers the model can add directly to a token's embedding.
