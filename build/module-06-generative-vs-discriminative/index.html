Steps to interact

  1. Click the "Discriminative" / "Generative" toggle.
  2. Compare the boundary-line view against the density-contour view.
Discriminative models learn the boundary between classes. Generative models learn the distribution of each class.
The discriminative view only draws where the line is. The generative view shows what each class actually looks like — notice it can tell you how confident it is even far from the line.
