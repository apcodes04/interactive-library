Steps to interact

  1. Drag the "Model size" slider.
  2. Watch the dot-grid grow.
  3. Watch capabilities unlock on the checklist as you cross each threshold.
At low scale, the model has few parameters and limited capabilities.

Emergent Capabilities

Basic text completion
Coherent multi-turn chat
Multi-step reasoning
Tool use / function calling
Capabilities like reasoning and tool use don't appear gradually — drag past each threshold and notice they switch on all at once, past a certain scale.
